# [距离的拥抱](https://hweinuo.github.io/)

东京与巴黎之间，隔着九千公里的距离和八小时的时差。

莉娜在巴黎的清晨醒来时，健太郎的东京正步入黄昏。他们的爱情活在手机屏幕和倒计时器中，数字一天天减少，直到下一次相见。

“今天代代木公园的樱花开始落了。”健太郎发来视频，花瓣如粉色雪花飘落在他肩头。

莉娜靠在巴黎公寓的窗边，窗外是灰蓝的晨空。“罗丹博物馆花园里的玫瑰刚刚结苞。”她翻转镜头，“看，这朵像不像我们上次视频时你画的那朵？”

他们相识于一次国际艺术展，莉娜是法国策展人，健太郎是日本画家。三天展览，却让他们决定用一生去了解彼此。

距离教会了他们新的表达方式。健太郎学会在深夜作画，为了让莉娜在巴黎的早晨第一时间看到；莉娜则收集巴黎各处的光影，做成电子相册，填满健太郎通勤的电车时光。

“我发明了一个新游戏，”一天视频时，健太郎说，“猜猜我今天想到了你几次。”

莉娜笑了：“每当我呼吸的时候？”

“更频繁些。”他温柔地说，“每次心跳之间。”

思念变成了一种创造性的力量。健太郎开始创作一系列名为“时差”的画作，画中巴黎与东京的风景奇妙地交融；莉娜则策划了一场关于“距离与亲密”的展览，展出来自世界各地的异地恋情侣的故事与信物。

最艰难的是生病的时候。去年冬天，莉娜患了重感冒，独自在公寓里发烧。健太郎在视频另一端束手无策，最后只能整夜开着视频，轻声给她读宫泽贤治的童话，直到她睡着。

“有时我觉得我们的爱情像月球，”莉娜曾在日记中写道，“总有一半在黑暗中，却依然用引力牵引着潮汐。”

终于，健太郎的个展在巴黎开幕，莉娜的展览也将在东京举行。机场相见的那一刻，他们没有像电影里那样奔跑相拥，只是静静走向对方，仿佛穿越的不是海关，而是无数个独自醒来的清晨和深夜。

“我带来了樱花。”健太郎从包里取出一个玻璃瓶，里面装着干燥的花瓣。

“而我保存了巴黎的雨。”莉娜递给他一个小瓶，标签上写着日期——他们第一次分别的那天。

距离没有稀释他们的爱情，反而像浓缩的星光，在黑暗中格外明亮。因为他们懂得，真正的拥抱不在于身体的接近，而在于心的同步跳动，哪怕相隔九千公里和八小时的时差。
